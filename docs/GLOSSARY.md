# Glosario de términos y abreviaturas

Este glosario centraliza términos recurrentes en las especificaciones y el roadmap.
Cuando se añadan nuevas pantallas o flujos, incorpora aquí las definiciones para mantener consistencia.

- Cómo usarlo: enlaza el término desde `docs/SPECS.md` o el README cuando aparezca por primera vez. Si un término cambia, actualiza este archivo antes de modificar otras referencias.

## CTA (Call To Action)

Botón principal o acción directa que el usuario debe realizar en una pantalla o tarjeta.

Ejemplos típicos de CTA:

- Solicitar plaza
- Cancelar solicitud
- Ser conductor
- Renunciar
- Ver detalle
- Completar viaje
- Marcar llegada

Si lees “CTA principal contextual”, es el botón que debe aparecer en una tarjeta según el estado de la solicitud y el rol del usuario.

## Salida

Instancia concreta de una lanzadera en una fecha y hora.

## Badge

Pequeño indicador visual de estado (colores verde, rojo, azul, gris, etc.).
