# CI screenshots

Capturas de estado de los PR para documentar el flujo de CI:


- `pr-checks-pending.png`: pantalla de PR con checks aún en progreso (“Some checks haven’t completed yet”).
- `pr-checks-green.png`: pantalla de PR con checks completados en verde (“All checks have passed”).

- Sirven como guía rápida: si el PR está en estado pendiente, no se debe hacer merge; solo mergea con el check verde.
